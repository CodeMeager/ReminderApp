# Мини-сайт «Напоминалка»

Простое веб‑приложение на чистом **HTML/CSS/JavaScript** для создания небольших напоминаний: указываете текст события, дату и время, отмечаете выполненные.

## Возможности

- Добавление напоминаний с полем **«Событие»** и **датой/временем**
- Отметка напоминания как **выполненное/невыполненное**
- Очистка всех **выполненных** напоминаний одной кнопкой
- Удобный современный тёмный интерфейс
- Автоматическое сохранение всех напоминаний в **localStorage** браузера

## Как запустить

### Вариант 1. Просто открыть файл

1. Откройте файл `index.html` двойным кликом в браузере.
2. Сайт сразу будет работать локально.

### Вариант 2. Через небольшой локальный сервер (npm)

1. Убедитесь, что у вас установлен **Node.js**.
2. В корне проекта выполните:

```bash
npm install
npm run start
```

3. Откройте указанный адрес в браузере (обычно `http://localhost:3000` или `http://localhost:5000` — `serve` напишет точный URL в консоль).

## Структура проекта

- `index.html` — разметка приложения.
- `style.css` — стили с современным тёмным оформлением.
- `script.js` — логика добавления, отображения и сохранения напоминаний.
- `package.json` — минимальная конфигурация npm (скрипт для запуска локального сервера).
